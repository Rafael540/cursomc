# 🛒 E-commerce API

Neste projeto desenvolvi uma **API REST de um sistema de e-commerce** utilizando **Java e Spring Boot**.

A aplicação simula o funcionamento de uma loja virtual, permitindo o gerenciamento de **categorias, produtos, clientes, pedidos e pagamentos**.

O objetivo deste projeto foi **praticar conceitos importantes de desenvolvimento backend**, como modelagem de domínio, arquitetura em camadas e persistência de dados com **JPA/Hibernate**.

## 🚀 Tecnologias que utilizei

- Java  
- Spring Boot  
- Spring Data JPA  
- Hibernate  
- Maven  
- H2 Database  

## 📚 Conceitos que pratiquei

Durante o desenvolvimento deste projeto trabalhei com:

- **Programação Orientada a Objetos (POO)**
- **Criação de APIs REST**
- **Relacionamentos entre entidades com JPA**
- **Arquitetura em camadas (Controller, Service e Repository)**
- **Persistência de dados com Hibernate**

## ▶️ Como executar o projeto

1. Clone o repositório

```

git clone [https://github.com/Rafael540/cursomc.git](https://github.com/Rafael540/cursomc.git)
```
